 </span> <span style="font-family: Helvetica-Bold; font-size: 18.0;text-align: left;">
     **Company Communication Policy**

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * Slack must be used for all internal written and audio based business communications.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * Zoom (beta) must be used for all enternal video based communications.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * When possible given the participants, Zoom (beta) must be used for external conferencing sessions be it audio or video.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * eMails should ideally be limited to communications with 3rd parties.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * Use of iMessage or FaceTime is allowed only when necessary and after initially attempting to use Slack.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * Use of any other messaging app or system is not allowed for company related communications

 </span>
 <span style="font-family: Helvetica-Bold; font-size: 18.0;text-align: left;">
     **IT & File Management**

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * All users must use inSync on their computers to manage file synchronization.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * Files should only be store locally when implemented through inSync.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * All files related to the company and the work should be on our Google Drive system and shared using Slack.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     * File sharing with 3rd parties must be handled using our Google Drive system.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     	* Folders modified for sharing with users outside the organization must be clearly labeled.

 </span> <span style="font-family: Helvetica; font-size: 14.0;text-align: left;">
     

I have been updating our security settings on Slack which might have lead with some annoyances logging into Slack.
We now have 2 factor authentication which requires each user to use a text message to send a code providing an additional level of security when logging into Slack. This feature will remain in effect but please tell me if any of you are having issues with it.
The other change is a forced logout function which mitigates the risk of any user unintentionally leaving an open connection. This was originally set for 24hrs and it is now changed to 7 days. Please share if you have any issues. 
We will be updating and sharing our Company Communication policy as well as our IT and File Management policy.
